# DebtShield AI

An Explainable Machine Learning Platform for Predicting Financial Distress, Poverty Risk, Housing Instability, and Economic Vulnerability.

## Overview

DebtShield AI integrates multiple public datasets to identify communities at risk of financial distress.

The platform analyzes housing costs, debt burdens, unemployment, wages, food access, energy affordability, and social vulnerability indicators to generate a Financial Distress Index and risk classification.

## Data Sources

- American Community Survey (ACS)
- Zillow
- Eviction Lab
- Federal Reserve Survey of Consumer Finances
- Bureau of Labor Statistics (BLS)
- MIT Living Wage Calculator
- CDC Social Vulnerability Index
- FHFA House Price Index
- USDA Food Access Research Atlas
- DOE LEAD Energy Affordability Data
- New York Fed Consumer Credit Data

## Features

- Financial Distress Index
- Housing Stress Analysis
- Debt Stress Analysis
- Food Access Risk Assessment
- Energy Burden Analysis
- Community Risk Classification
- Policy Recommendations
- Interactive Dashboard

## Technology Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- Streamlit
- Matplotlib

## Project Goal

To help policymakers, researchers, nonprofits, and communities identify financial vulnerability before crises occur.
